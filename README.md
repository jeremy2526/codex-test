# 前端练习 Demo 合集 🎓

这是一组用来练习 **HTML / CSS / JavaScript** 的小工具和小游戏。
每个文件都是**单独的、可直接运行**的 HTML 文件，代码里有**详细的中文注释**，适合边玩边学。

## 🚀 如何运行

不需要安装任何东西，直接用浏览器打开即可：

1. 在文件管理器里**双击 `index.html`**，或
2. 把 `index.html` 拖进浏览器窗口

然后从首页点击进入各个 demo。

## 📂 文件说明

| 文件 | 内容 | 主要练习点 |
| --- | --- | --- |
| `index.html` | 导航首页 | 链接跳转、CSS 卡片布局 |
| `guess-number.html` | 🎯 猜数字游戏 | 随机数、条件判断、事件监听 |
| `todo.html` | ✅ 待办清单 | 数组增删、动态渲染、本地存储 |
| `calculator.html` | 🧮 简易计算器 | 字符串处理、事件委托、`data-*` 属性 |

## 💡 建议的练习方式

1. **先玩** —— 打开页面，把每个功能都点一遍，感受它做了什么。
2. **再读** —— 用编辑器打开对应的 `.html`，对照注释看每一段代码。
3. **动手改** —— 试试下面这些小挑战：
   - 猜数字：把范围改成 1~50，或限制最多只能猜 7 次。
   - 待办清单：加一个「清除所有已完成」的按钮。
   - 计算器：加一个「百分比 %」或「正负号 ±」按钮。

改完保存，刷新浏览器就能立刻看到效果——这就是前端最爽的地方 😎

## ⚠️ 小提示

`calculator.html` 里为了教学简单，用了 `Function()` 来计算算式。
这种写法在真实项目中处理用户输入时需要更严格的校验，这里仅作演示。
